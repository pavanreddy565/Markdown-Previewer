# Markdown Previewer

This project is a Markdown Previewer developed using ReactJS. It allows users to write and preview Markdown syntax in real-time. The editor is designed to be responsive, ensuring a seamless experience across various devices.

## Features

1. **Real-time Preview:** The Markdown editor provides a live preview of the rendered HTML as you type, allowing users to see how their Markdown content will appear.

2. **Responsiveness:** The editor is optimized for responsiveness, ensuring a consistent and user-friendly experience on desktop and mobile devices.

3. **Easy to Use:** The interface is designed for simplicity and ease of use, making it accessible to users of all skill levels.

## Usage

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/pavanreddy565/markdown_editor.git
    ```

2. **Navigate to Project Directory:**
    ```bash
    cd markdown-editor
    ```

3. **Install Dependencies:**
    ```bash
    npm install
    ```

4. **Start the Development Server:**
    ```bash
    npm start
    ```

5. **Open in Browser:**
   - Open [http://localhost:3000](http://localhost:3000) in your web browser to view the Markdown editor.
   
6. **Editing Markdown:**
   - Type your Markdown content in the editor pane.
   - See the live HTML preview in the preview pane.

7. **Save and Share:**
   - Once you're satisfied with your Markdown content, you can save it or copy the generated HTML code for sharing.

## Dependencies

This project uses the following external library for converting Markdown to HTML:
- [marked.js](https://cdnjs.cloudflare.com/ajax/libs/marked/12.0.2/marked.min.js)
